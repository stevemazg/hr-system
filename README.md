# HR System

Self-hosted HR management platform — holiday, contracts, absence, wage tracking.

## Features
- Multi-organisation support
- Holiday allowance & leave management (carry-forward, adjustments)
- Employee profiles, contracts, personal data
- Sick & absence tracking
- Wage tracking (admin only)
- Document vault
- Role-based access: Global Admin / Manager / User

## Stack
Laravel 11 / PHP 8.2 + MariaDB + Tailwind CSS + Alpine.js on AWS Lightsail (eu-west-2)
